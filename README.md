# ColorProcessor.js
## 概要
自分のサイト作成に必要になったので作りました

バグだらけです()
## 関数: colorMerge(A, B)
色の結合が出来ます

| 引数 | 内容 |
| ---- | ---- |
| A | 一つ目の色 |
| B | 二つ目の色 |

注) #を付けても処理できますが、推奨しません

注) #HexColorの形式のみ対応しています。rgb(0,0,0)は使えません

注) 現時点で10進数しか返せません

記) #000の様に三桁でも強制的に六桁にして計算できます
## ライセンス
